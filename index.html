Manual for anim (PDF format)

To use this, download to any directory on your computer. Unpack by changing to that directory and running

tar xvf anim.tar

The manual including installation instructions is included in the tarball.

In short, you may install on a Mac by running make anim-mac and make demos. Because OSX has formally deprecated OpenGL 2, you will get compiler warnings, but these shouldn't cause a problem. Copy the resulting anim binary to somewhere on your system path; you can do this automatically with sudo make install.

You may install on Linux by installing the freeglut3-dev and libglew-dev packages, then running make anim and make demos. Copy the resulting anim binary to somewhere on your system path; you can do this automatically with sudo make install.
